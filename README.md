# MPS_format




## Installation

1. Clone this repository:   
    ```
    git clone https://github.com/Nick-Liou/MPS_format.git
    ```
2. Navigate to the project directory:   
    ```
    cd MPS_format
    ```
3. Install the required dependencies using pip:
    ```
    pip install -r requirements.txt
    ```
